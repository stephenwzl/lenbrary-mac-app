# 02 — 设计语言

## 设计原则

1. **照片是主角** — UI 退到背景里，不抢内容
2. **丢进来就行** — 交互极简，不给用户做选择题
3. **后台默默处理** — 不弹窗、不阻塞、不打扰

## 主题色（莫兰迪色系）

低饱和、不抢照片、亮暗模式都好看。

| 角色 | 色号 | 用途 |
|------|------|------|
| 主色 | `#8E9AAF` | 选中态、按钮、进度条、搜索高亮 |
| 好片/成功 | `#A7B5A0` | 好片标记、成功状态 |
| 废片/删除 | `#C09080` | 废片标记、删除按钮 |
| 背景（亮） | `#F2EFEA` | 亮模式背景 |
| 背景（暗） | `#2A2A2E` | 暗模式背景 |
| 文字主色 | `#4A4A50` | 标题、主要内容 |
| 文字次色 | `#9A9A9E` | 辅助文字、描述 |
| 分隔线 | `#E0DCD6` | 边框、分隔 |

## 字体

- 不自定义字体

## 界面布局

### 三栏结构

```
┌──────────────────────────────────────────────────────────────┐
│  🔍 搜索照片...（自然语言搜索）                               │ ← 顶部搜索栏
├─────────┬────────────────────────────────────────┬───────────┤
│ 侧边栏  │              照片网格                   │ 照片详情  │
│         │                                        │           │
│ 导入    │   缩略图网格（无限滚动）                │ EXIF     │
│ 设备列表│   按日期分组                            │ AI 描述  │
│         │                                        │ 质量评分  │
│ 存储    │                                        │ 操作按钮  │
│ 本地    │                                        │           │
│ WebDAV  │                                        │           │
│ S3      │                                        │           │
├─────────┴────────────────────────────────────────┴───────────┤
│  后台处理状态                                                 │ ← 底部状态栏
└──────────────────────────────────────────────────────────────┘
```

### 各区域规范

**顶部搜索栏**
- 常驻最醒目位置，Cmd+F 聚焦
- 自然语言输入，实时搜索
- 搜索结果替换网格内容
- ESC 或 × 清空回到全部照片

**左侧边栏**
- 导入区（动态）：自动检测设备，显示设备名 + 新照片数
- 存储区（静态）：本地/WebDAV/S3 状态
- 无分类相簿

**照片网格**
- LazyVGrid，缩略图自适应列数
- 按日期分组，组间日期分隔条
- 选中高亮边框（主色）
- 索引中显示微小 loading 角标

**右侧详情面板**
- Cmd+I 切换显示/隐藏
- 大缩略图 + EXIF + AI 描述 + 质量评分
- 操作：标记好片（孔雀绿）、标记废片（波尔多红）、删除

**底部状态栏**
- 始终可见，一行
- 后台处理进度（索引/去重/EXIF）
- 无任务时显示照片总数 / 存储占用

## 状态指示

| 状态 | 视觉 |
|------|------|
| 照片选中 | 主色 `#8E9AAF` 边框 |
| 正在索引 | 微小 loading 角标 |
| 已标记好片 | 孔雀绿 `#A7B5A0` 角标 + ⭐ |
| 已标记废片 | 陶土色 `#C09080` 角标 |
| 重复照片 | 重叠图标 |
| 设备已连接 | 侧边栏正常显示 |
| 设备导入中 | 侧边栏进度条 |
| 存储已连接 | 绿色状态点 |
| 存储未配置 | 灰色状态点 |

## 交互原则

- **不弹大窗口** — 所有操作内联，不打断用户
- **后台不阻塞** — 导入完立即可浏览，索引后台跑
- **AI 搜索替代分类** — 找东西靠搜，不靠手动整理
- **辅助决策不替代决策** — AI 排序/评分，用户拍板

## 技术要点

- SwiftUI `NavigationSplitView`（三栏布局）
- SwiftUI `LazyVGrid`（照片网格）
- `ImageCaptureCore`（设备检测和导入）
- 后台任务 `Task` + SQLite 进度持久化
- 缩略图缓存：DiskCache + MemoryCache 双层
