# 01 — 产品概览

## Vision

**PhotoVault** — 你的照片终点站。

用户的核心动作就一个：**把照片丢进来**。丢完就不用管了。PhotoVault 负责备份、去重、索引、整理。用户什么时候想找，打开就能搜到。

## 问题

摄影师和摄影爱好者的照片管理存在三个真实痛点：

1. **跨设备** — iPhone、相机、Android 手机，生态各不相同，没有统一入口
2. **太多重复** — 同一个场景拍了几十张，筛选耗时耗力
3. **找不到** — 照片堆在硬盘里，没有可搜索的索引

现有工具的问题：
- Apple/Google Photos：数据在云端，不可控
- Lightroom：专业但重，不解决"找"的问题
- Immich：自托管但需要技术门槛

## 解决方案

一个 Mac App，后台默默处理一切：

```
照片进来 → 备份（不会丢）→ EXIF 提取 → 去重 → AI 索引（描述+评分）→ 可搜索
```

用户只需要做两件事：**丢照片** 和 **搜索照片**。

## 目标用户

| 用户类型 | 特征 | 核心需求 |
|----------|------|----------|
| 摄影师 | 多设备、大量照片、需要筛选交付 | 筛选省时间 |
| 摄影爱好者 | 设备多、照片散落 | 统一管理 |
| 照片收藏者 | 长期积累、想回溯 | 能找到 |

## 技术方案

- **检索**：ripgrep + SQLite FTS5（不用向量数据库）
- **AI 索引**：多模态 API 批量处理（GPT-4o Vision / Claude Vision），索引一次后搜本地
- **去重**：EXIF 时间戳 + pHash 感知哈希
- **存储**：文件系统（按日期/设备分目录）+ SQLite（元数据）
- **前端**：SwiftUI 原生 Mac App
