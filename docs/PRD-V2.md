# 04 — PRD: 后续版本规划

## v0.2 — 设备导入

**目标**：照片不用先整理到文件夹，插上就能导。

### F11 — 设备自动检测

- 使用 macOS `ImageCaptureCore` 框架
- 自动检测 iPhone、Android 手机、相机、SD 卡
- 侧边栏动态显示已连接设备 + 新照片数量
- 支持"导入全部"和"仅导入新照片"

### F12 — 设备导入流程

- 点击设备 → 选择导入范围 → 开始导入
- 导入进度内联显示在侧边栏，不弹窗
- 导入完成后设备项变灰
- 导入的文件自动进入 EXIF 提取 → 去重 → AI 索引 流程

## v0.3 — 筛选增强

**目标**：从"能找到"升级到"帮你看"。

### F13 — 重复照片组一键清理

- 去重完成后，重复组可展开查看
- 显示"同场景 X 张，AI 推荐保留这张"
- 一键"保留最佳，删除其余"
- 删除前确认

### F14 — 质量评分排序

- 按 AI 质量评分排序网格
- 可筛选：只显示高分照片 / 只显示低分照片
- 批量标记低分为废片

### F15 — 大图查看模式

- 双击或回车进入全屏/近全屏查看
- 左右箭头 / 滑动切换
- 底部显示"第 X 张 / 共 Y 张"
- 快捷键：← → 切换、⭐ 标记好片、Delete 标记废片、ESC 退出

### F16 — 时间线筛选

- 侧边栏年 → 月两级折叠
- 点击某月，网格只显示该月照片
- 搜索 + 时间筛选可叠加

## v0.4 — 存储扩展

**目标**：照片不只存本地。

### F17 — WebDAV 存储后端

- 配置 WebDAV 服务器地址 + 认证
- 照片上传到 WebDAV
- 本地保留缩略图，原图可从 WebDAV 拉取
- 存储状态显示在侧边栏

### F18 — S3 兼容存储后端

- 支持 AWS S3、Cloudflare R2、MinIO 等 S3 兼容服务
- 配置 endpoint / bucket / access key / secret key
- 照片上传 + 缩略图本地缓存

### F19 — 存储管理

- 存储用量统计（本地 / WebDAV / S3）
- 存储路径配置
- 存储健康检查

### F20 — 导出

- 选中照片导出到指定文件夹

- 可选导出 AI 描述（生成 sidecar 文件或嵌入 EXIF）
